# Paymenter Market

Paymenter market source code, it contains users/resources and more!

**! NO SUPPORT PROVIDED ON HOW TO SET THIS UP OR ANYTHING !**


# How to run:

## Requirements:
- NodeJS
- Composer
- MySQL
- PHP

## Setup:
- Clone the repo
- Run `composer install`
- Run `npm install`
- Run `npm run build`
- Run `php artisan migrate`
